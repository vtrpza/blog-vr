# blog.vradvogados.com.br Lead Engine

Motor orgânico de aquisição para a VR Advogados: Astro + TypeScript + MDX + Tailwind, com arquitetura preparada para Cloudflare Pages/Workers/D1, Turnstile, Pipedrive e tracking pesado.

## Metodologia

- Implementação por slices verticais.
- TDD para regras e contratos reutilizáveis.
- Build/check/test antes de considerar fase concluída.
- Sem promessas jurídicas, sem publicação automática por IA.

## Comandos

```bash
npm install
npm test
npm run check
npm run build
```
